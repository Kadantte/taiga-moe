Comparison of Anime Library APIs

This page aims to draw a comperative picture of Hummingbird, Kitsu and MyAnimeList APIs. I first put this together in order to aid the development of Taiga, but hopefully it will also help other developers and serve as an indicator of each API's shortcomings.

A couple of things to note before venturing forth:

  • Some methods use different names to refer to the same value (e.g. title and series_title).
  • If an API is missing a value that others provide, it's marked as n/a. If the data is valuable and should be included in the API, it's red; otherwise it's gray.
  • Hummingbird has been rebranded as Kitsu in December 2016.

For more information on each API, please refer to their official documents:

  • Hummingbird: API documentation, Forum category
  • Kitsu: API documentation
  • MyAnimeList: API documentation, Club page
